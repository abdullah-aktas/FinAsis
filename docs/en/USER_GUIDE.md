## Introduction

FinAsis is a comprehensive financial management system for businesses. This guide explains the basic features and usage of the system.

## System Requirements

- Web browser (Chrome, Firefox, Safari, Edge)
- Minimum screen resolution: 1280x720
- Internet connection

## Account Management

### Logging In
1. Open FinAsis website
2. Enter your username and password
3. Click "Login" button

### Changing Password
1. Go to Profile menu
2. Select "Change Password"
3. Enter current password
4. Set new password
5. Save changes

### Accounting Module

## Reporting

### Creating Reports
1. Go to Reports menu
2. Select desired report type
3. Set date range
4. Click "Generate Report" button

### Report Export
- PDF format
- Excel format
- CSV format
